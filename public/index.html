API Films

Cette API permet d'afficher, de modifier et de supprimer des films de la base de donnees.

l'accès aux routes:

https://api-films-dsr0.onrender.com/

Github:

github.com/guillovilla

Routes:

Récupérer la liste de tous les films

GET/api/films/

https://api-films-dsr0.onrender.com/api/films

Paramètres possibles :

http://localhost:3301/api/films?tri=annee&order-direction=desc

tri

  • titre
  • description
  • annee
  • realisation

order-direction

  • asc
  • dsc

Récupérer un film spécifique

GET/api/films/:id

https://api-films-dsr0.onrender.com/api/films/PGmeX8SXuFKucv0Qs9Lp

Ajouter un nouveau film

POST/api/films/

Paramètres requis : Un objet film dans le corp de la requête

const film = {
titre:"Le silence des agneaux",
genres:["Crime","Drame","Thriller"],
description:"Une jeune recrue du FBI interroge un psychopathe cannibale afin de l'aider à traquer un autre tueur en série.",
titreVignette":"lesilencedesagneaux.jpg",
realisation":"Jonathan Demme",
annee:1991
}

Modifier un film spécifique

PUT/api/films/:id

Paramètres requis : Informations à modifier du film dans le corp de la requête

{
titre:"Le silence des agneaux",
annee:1991
}

Supprimer un film spécifique

DELETE/api/films/:id
